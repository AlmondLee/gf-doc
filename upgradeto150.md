# `1.x`升级到`v1.5.0`

1. 代码主库从`gitee`迁移到了`github`。因此需要在您的代码中做以下全局替换：

    ```
    gitee.com/johng/gf
    ```
    替换为
    ```
    github.com/gogf/gf
    ```

2. 模块`gregex`和`gstr`从`util`分类迁移到了`text`分类。因此需要在您的代码中做以下全局替换：
    ```
    /util/gstr
    /util/gregex
    ```
    替换为
    ```
    /util/gstr
    /text/gregex
    ```
